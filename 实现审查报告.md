### 1. 推送功能未完整实现
sendPush控制器中标记了TODO，实际推送逻辑需要对接推送服务（APNs/FCM/华为推送等）。

### 2. 部分统计功能缺失
数据统计报表中的用户留存率、用户分布、付费转化率、ARPU等高级统计未实现。

### 3. 数据库迁移需要执行
新增的数据库表需要在服务器上执行迁移脚本：
```bash
mysql -u yujian -p'Yujian@2024DB' yujian < src/db/add_features_batch2.sql
mysql -u yujian -p'Yujian@2024DB' yujian < src/db/add_admin_features.sql
```

### 4. WebSocket信令需要更新
新增的功能（游戏、亲密关系升级通知等）可能需要在websocket-server.js中添加新的消息类型处理。
